# minecraft-server
A repository to consolidate config files, updates, and any other important information for my (private) Minecraft server.

![Server Spawn](https://github.com/zanemooney/minecraft-server/blob/main/imgs/serverSpawn.png?raw=true)

## Install Guide

Tiger's  Modpack
1. Download `.zip` by clicking `Tiger's Modpack v2.5.zip` in file list, then clicking "Download Raw File" (download icon).
![Download Help](https://github.com/zanemooney/minecraft-server/blob/main/imgs/downloadStep.png?raw=true)
3. Open CurseForge
4. In the Minecraft section (CurseForge), under "My Modpacks," click **Import**
![Download Help](https://github.com/zanemooney/minecraft-server/blob/main/imgs/modInstallStep2.png?raw=true)
6. Import Profile `.zip`
7. Select `.zip`, allow CurseForge to download mods
8. Launch Minecraft using the modpack
9. *(Optional)* Enable shaders (Video Settings in-game)
10. Join the server!

No world backup included. =( The world size is over 3 GB. DM me for the file if you want.
