## 🚀 Install Guide: Tiger's Modpack

Follow these steps to get the modpack running:

### 1️⃣ Download the Modpack
Download the `.zip` by clicking `Tiger's Modpack v2.5.zip` in the file list, then click **Download Raw File** (download icon).  

<div align="center">
  <img src="https://github.com/zanemooney/minecraft-server/blob/main/imgs/downloadStep.png?raw=true" alt="Download Step" style="max-width:70%; border:1px solid #ccc; border-radius:6px;">
  <p style="font-size:0.9em; color:gray;">Click the download icon to get the ZIP file</p>
</div>

### 2️⃣ Open CurseForge
Open the CurseForge launcher and navigate to **Minecraft → My Modpacks**.

### 3️⃣ Import the Modpack
Click **Import** and select the downloaded `.zip`.  

<div align="center">
  <img src="https://github.com/zanemooney/minecraft-server/blob/main/imgs/modInstallStep2.png?raw=true" alt="Mod Install Step 2" style="width:50%; border:1px solid #ccc; border-radius:6px;">
  <p style="font-size:0.9em; color:gray;">Import the ZIP file into CurseForge</p>
</div>

### 4️⃣ Launch the Game
- Select the `.zip` profile in CurseForge and let it download all mods.  
- Launch Minecraft using the modpack.

### 5️⃣ Optional: Enable Shaders
Enable shaders via **Video Settings → Shaders** for a prettier experience.

### 6️⃣ Join the Server
Connect using the **IP in the Discord server**. *(Not included in the modpack for privacy.)*
